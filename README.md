# HDG Smart-Calc
**Hot Dip Galvanizing Quality Control Management System**

## 🚀 Overview
A digital solution designed to replace error-prone Excel sheets in the galvanizing process. This application manages the chemical balance and dosing specifically for the **Degreasing** phase of the galvanizing line.

## ✨ Key Features
* **Chemical Calculator:** Precise dosing logic for degreasing tanks.
* **Dashboard:** Real-time overview of current bath parameters.
* **History & Logs:** Track past dosing operations for audit and analysis.
* **Knowledge Base:** Integrated documentation for QC standards.
